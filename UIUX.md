# UI/UX 디자인 가이드
## 원가관리 웹 시스템 디자인 명세

### 1. 디자인 철학

#### 1.1 핵심 원칙
- **미니멀리즘**: 군더더기 없는 깔끔한 인터페이스
- **데이터 중심**: 숫자와 정보가 명확하게 보이도록
- **효율성**: 최소 클릭으로 원하는 정보에 도달
- **직관성**: 별도 학습 없이 바로 사용 가능

#### 1.2 사용자 경험 목표
- 3초 이내에 원하는 정보 찾기
- 원클릭으로 핵심 기능 접근
- 시각적 피로도 최소화
- 모든 액션에 즉각적인 피드백

### 2. 컬러 시스템

#### 2.1 메인 컬러 팔레트
```css
/* Primary Colors */
--primary-blue: #2563eb    /* 메인 액션 버튼 */
--primary-dark: #1d4ed8    /* 호버 상태 */
--primary-light: #dbeafe  /* 배경, 강조 */

/* Semantic Colors */
--success-green: #059669   /* 높은 마진율 */
--warning-yellow: #d97706  /* 보통 마진율 */
--danger-red: #dc2626     /* 낮은 마진율, 삭제 */

/* Neutral Colors */
--gray-900: #111827       /* 제목, 중요 텍스트 */
--gray-600: #4b5563       /* 본문 텍스트 */
--gray-400: #9ca3af       /* 보조 텍스트 */
--gray-100: #f3f4f6       /* 배경, 구분선 */
--white: #ffffff          /* 카드, 모달 배경 */
```

#### 2.2 컬러 사용 규칙
- **높은 마진율 (70% 이상)**: 초록색 표시
- **보통 마진율 (40-70%)**: 노란색 표시  
- **낮은 마진율 (40% 미만)**: 빨간색 표시
- **배경**: 연한 회색 (#f8fafc)으로 통일

### 3. 타이포그래피

#### 3.1 폰트 시스템

```css
/* Font Family */

/* Font Sizes */
```

#### 3.2 텍스트 계층구조

3. **카드 제목**: 18px, Medium, Gray-900
4. **본문 텍스트**: 16px, Normal, Gray-600
5. **보조 텍스트**: 14px, Normal, Gray-400

### 4. 레이아웃 시스템

#### 4.1 그리드 시스템
- **컨테이너 최대폭**: 1200px
- **그리드**: 12컬럼 시스템
- **간격**: 16px (모바일), 24px (데스크톱)
- **여백**: 패딩 20px (모바일), 40px (데스크톱)

#### 4.2 반응형 브레이크포인트


### 6. 데이터 시각화

#### 6.1 수치 표시 방법
- **금액**: 천 단위 쉼표 + "원" 단위 표시 (예: "1,234,567원")
- **퍼센트**: 소수점 1자리 + "%" (예: "85.2%")
- **마진율 색상**: 수치에 따른 색상 자동 적용
- **변화율**: 증감 화살표 아이콘 표시

#### 6.2 표(Table) 디자인
```css
.table {
  width: 100%;
  border-collapse: collapse;
  background: white;
  border-radius: 8px;
  overflow: hidden;
}

.table th {
  background: #f9fafb;
  padding: 12px 16px;
  text-align: left;
  font-weight: 600;
  color: #374151;
}

.table td {
  padding: 16px;
  border-top: 1px solid #e5e7eb;
}
```

### 7. 인터랙션 가이드

#### 7.1 호버 효과
- **카드**: 살짝 위로 이동 + 그림자 강화
- **버튼**: 색상 변화 (primary → primary-dark)
- **행**: 연한 배경색으로 하이라이트

#### 7.2 로딩 상태
- **스켈레톤 UI**: 콘텐츠 로딩 중 회색 박스 애니메이션
- **스피너**: 버튼 클릭 시 작은 스피너 표시
- **프로그레스 바**: 데이터 업로드/처리 시

#### 7.3 피드백 시스템
- **토스트 메시지**: 성공/실패/정보 메시지
- **모달**: 중요한 액션 확인
- **인라인 유효성 검사**: 실시간 입력 검증

### 8. 페이지별 상세 디자인

#### 8.1 대시보드 페이지
- **KPI 카드**: 4개 카드 (총 시술수, 평균 마진율, 최고 마진, 추천 제품수)
- **차트 영역**: 2개 차트를 좌우 배치
- **빠른 액션**: 새 시술 추가, 재료 관리 바로가기 버튼

#### 8.2 시술 관리 페이지  
- **검색 바**: 상단에 시술명/카테고리 검색
- **필터링**: 카테고리별, 마진율 범위별 필터
- **정렬**: 이름순, 가격순, 마진순 정렬 옵션
- **테이블**: 페이지네이션이 있는 데이터 테이블

#### 8.3 재료 관리 페이지
- **간단한 목록**: 재료명과 원가만 표시
- **인라인 편집**: 클릭하면 바로 수정 가능
- **빠른 추가**: 하단 고정 입력 폼
- **대량 업로드 버튼**: 우측 상단에 "Excel 업로드" 버튼

### 9. 접근성 가이드

#### 9.1 키보드 내비게이션
- Tab 키로 모든 인터랙션 요소 접근 가능
- Enter/Space 키로 버튼 활성화
- ESC 키로 모달/드롭다운 닫기

#### 9.2 스크린 리더 지원
- 적절한 ARIA 라벨 사용
- 의미 있는 alt 텍스트
- 폼 요소와 라벨 연결

#### 9.3 색상 접근성
- 명도 대비 4.5:1 이상 유지
- 색상 외에 아이콘/텍스트로도 정보 전달
- 색맹 사용자 고려한 색상 선택

### 12. 모바일 최적화

#### 10.1 터치 친화적 디자인
- 터치 타겟 최소 44px × 44px
- 적절한 터치 간격 유지
- 스와이프 제스처 지원

#### 10.2 모바일 레이아웃
- 단일 컬럼 레이아웃
- 중요 정보 우선 표시
